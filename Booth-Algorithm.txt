Booth Algorithm:
	y = -2^(n-1)*y(n-1) + 2^(n-2)*y(n-2) + ... + 2^(0)*y(0)
	  = -2^(n-1)*y(n-1) + 2^(n-2)*y(n-2) + 2^(n-2)*y(n-2) - 2^(n-2)*y(n-2) + 2^(n-3)*y(n-3) + 2^(n-3)*y(n-3) - 2^(n-3)*y(n-3) + ... +  2^(0)*y(0) + 2^(0)*y(0) - 2^(0)*y(0) +2^(-1)*y(-1)
	  = -2^(n-1)*(y(n-2)-y(n-1)) + 2^(n-2)*(y(n-3)-y(n-2)) + ... + 2^(0)*(y(-1)-y(0))

y(n-2)-y(n-1) = 1 乘数
	      = 0 0
	      =-1 乘数的补码
